
En este archivo se genera una documentación sobre el trabajo realizado. 
el cual es sobre una gestión de inventario el cual toma datos de libros
hay 8 funciones en las cuales.

1.se implementa la funcion addbooks que se usa para agregar libros
2.consultbooks permite mostrar los libros registrados
3.updatebooks, permite actualizar los libros
4.deletebooks, permite eliminar los libros
5.addsales, permite generar un historico de clientes
6.report, muestra los top más vendidos
